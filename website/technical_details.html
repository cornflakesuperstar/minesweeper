Build Notes -

My implementation of Minesweeper utilised the following technologies:

Java, HTTP, PHP, XML, Browser Cookies, JavaScript, HTML, CSS and an Apache Web Server on a UNIX box
In order to support the widest range of Web Browsers and Operating Systems, I decided to implement the front end in a Java Applet. Any Java compliant browser with a Java Runtime Environment of JRE 1.1 or greater should be able to run the game.

The game's User Interface was written entirely in Java (mostly utilising the Applet and AWT packages). I created several custom AWT Components that are reuseable and extendable for other purposes (see Downloads section below).

I imported Netscape's JavaScript package into the Applet in order to enable interaction with the client browser's Cookie Object to store user-specific High Scores.

The central High Scores list is maintained in an XML file on my Apache Web Server. I wrote a PHP script to modify the XML, making use of a pure PHP XML parser called "xml2Tree" written by Bill Softky (see Links section below) (My University refused to re-compile the PHP module on the Apache Web Server with XML support).

My Applet imports a freeware lightweight XML Parser written in Java called "NanoXML 2 Lite" created by Marc De Scheemaecker. I extended the functionality of the parser to include a node selecting method: selectSingleNode() (see Links section below).

Certain browser specific code was included in the Applet to accomodate for non-standard JVM's on the client's browser (this handles in particular, the Internet Explorer JVM's non-standard implementation of Event propagation and unusual Frame rendering behaviour).

My development environment consisted of a PII Celeron 333 running Windows 2000. I wrote all Java code using a shareware text editor called MED. I used release 1.3 of the Java Development Kit to compile and test my code (with Appletviewer). All HTML/JavaScript/CSS was written in Homesite version 4.5.2, and I wrote the PHP using vi on a UNIX account. This was eventually tested using Internet Explorer 5.5.
